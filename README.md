## Getting Started

### Prerequisites

- [Python >=3.11](https://www.python.org/downloads/)
- [Poetry >=1.6](https://python-poetry.org/)

### Install dependencies with Poetry:

```bash
poetry install
```

### Run the program:

```bash
poetry run poe cli

# or

poetry shell
poe cli
```
